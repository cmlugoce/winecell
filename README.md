# winecell

Track your favorite wines with Winecell!

# Installation and Usage

Fork it!

Clone it!

`cd winecell-api`

`rake db:seed`

`rails s -p 3001`

Open a new terminal tab and do the following:

`cd winecell-client`

`npm start`

All set!!
